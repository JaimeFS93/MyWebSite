javaScript Files

In this section we will present you all js files that are used in this template.

JavaScript files used are as follows:

  1. jQuery
  2. Isotope plugin
  3. waypoints.min.js - A library for providing animation support
  4. bxSlider - A simple yet powerfull gallery script
  5. ionrangeslider - A small slider that is used to shop pricing variations
  6. Countdown
  7. gmap3.min - for google maps

  8. Media Elements - Provides support for HTML5 video and audio

  9. Revolution Slider - A very powerfull slider with a lot of options
  10. Liquid - A slider designed for bouncing elements in specific manner
  11. main - the custom functions are in this file
