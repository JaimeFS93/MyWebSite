Color style

In this section we will present you and explain how to choose layout and color style.

There are six color options you can choose from. Of course you can use your custom made color. predefined colors are:

  1. orange.css - Default color style
  2. blue.css - Blue color style
  3. lightblue.css - Light blue color style
  4. darkgreen.css - Green color style
  5. red.css - Red color style
  6. yellow.css - Yellow color style
  7. color5.css - light green color style
  8. color6.css - light purple color style
  9. color7.css - light dark orange color style
  10. color8.css - light barn red color style

If you like to use your custom color, simply use one of the "color".css files and replace the color. So, to change to color style you need to replace the name of the style you are using in all HTML pages. If you choose your own color, you'll need to change the color of some icons - that is accordion-active.png and reply-hover icon. Icons that are in color of the style you are using are placed in img folder.
